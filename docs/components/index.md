### 快速上手

::: tip 提示

s-ui-plus 基于 vue3 + ts + Element-plus 再次封装的基础组件

:::

### 安装

```bash:no-line-numbers
npm add chriszj
&
npm  install chriszj
```


### 使用

```js
// main.ts
import Chriszj from "chriszj";
import "chriszj/style.css";
const app = createApp(App)
app.use(Chriszj)
app.mount('#app')
```
